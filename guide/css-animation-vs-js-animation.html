    # 当JS引擎执行时GUI线程会被挂起, 为什么css动画还在执行?

    查看这个演示 (opens new window), 在js执行期间, 更改transform属性的css动画一直在执行, 而更改width属性的css动画和js动画停止, 答案在下面的链接中。

    简单解释就是, 更改transform属性来实现动画时会避免布局和绘制，只需要合成更改。
